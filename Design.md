# 受 Linear 启发的视觉设计系统

## 1. 视觉主题与氛围

Linear 的网站是深色优先产品设计的典范——近乎纯黑的画布（`#08090a`）上，内容如星光般从黑暗中浮现。整体印象是极致的精密工程感：每个元素都存在于精心校准的亮度层级中，从几乎不可见的边框（`rgba(255,255,255,0.05)`）到柔和发光文字（`#f7f8f8`）。这不是将深色主题套用在浅色设计上——而是将黑暗作为原生媒介，通过白色透明度的微妙渐变而非色彩变化来管理信息密度。

字体系统完全基于 Inter Variable 构建，全局启用 OpenType 特性 `"cv01"` 和 `"ss03"`，赋予字体更干净、更几何化的特征。Inter 的字重范围极广——从 300（轻量正文）到 510（中等，Linear 的标志性字重）再到 590（半粗强调）。510 字重尤为独特：它介于常规和中等之间，创造出一种不张扬的微妙强调感。在展示尺寸（72px、64px、48px）下，Inter 使用激进的负字间距（-1.584px 至 -1.056px），打造出压缩、权威的标题，感觉像是工程产物而非设计作品。Berkeley Mono 作为代码和技术标签的等宽字体伴侣，备选字体包括 ui-monospace、SF Mono 和 Menlo。

色彩系统几乎完全是无色的——深色背景配白/灰文字——仅由一个品牌点缀色点缀：Linear 标志性的靛蓝紫（`#5e6ad2` 用于背景，`#7170ff` 用于交互强调）。这种强调色使用克制且有目的性，仅出现在 CTA、激活状态和品牌元素上。边框系统采用超细半透明白色边框（`rgba(255,255,255,0.05)` 至 `rgba(255,255,255,0.08)`），在无声中创造结构，如同月光下绘制的线框。

**关键特征：**
- 原生深色模式：`#08090a` 营销背景、`#0f1011` 面板背景、`#191a1b` 抬升表面
- 全局使用 Inter Variable 并启用 `"cv01", "ss03"`——几何变体带来更干净的视觉效果
- 标志性 510 字重（介于常规和中等之间）用于大部分 UI 文字
- 展示尺寸采用激进的负字间距（72px 时为 -1.584px，48px 时为 -1.056px）
- 品牌靛蓝紫：`#5e6ad2`（背景）/ `#7170ff`（强调）/ `#828fff`（悬停）——系统中唯一的彩色
- 全站使用半透明白色边框：`rgba(255,255,255,0.05)` 至 `rgba(255,255,255,0.08)`
- 按钮背景接近零透明度：`rgba(255,255,255,0.02)` 至 `rgba(255,255,255,0.05)`
- 多层阴影配合内嵌变体，为深色表面创造深度
- Radix UI 基础组件作为组件基础（检测到 6 个基础组件）
- 成功绿色（`#27a644`、`#10b981`）仅用于状态指示器

## 2. 色彩体系与角色

### 背景表面
- **营销黑**（`#010102` / `#08090a`）：最深的背景——主视觉区和营销页面的画布。近乎纯黑，带有难以察觉的蓝冷底色。
- **面板暗色**（`#0f1011`）：侧边栏和面板背景。比营销黑高一级。
- **三级表面**（`#191a1b`）：抬升表面区域、卡片背景、下拉菜单。
- **二级表面**（`#28282c`）：最浅的深色表面——用于悬停状态和轻微抬升的组件。

### 文字与内容
- **主要文字**（`#f7f8f8`）：近白色，带有微暖色调。默认文字颜色——非纯白，防止深色背景下的眼部疲劳。
- **次要文字**（`#d0d6e0`）：冷银灰色，用于正文、描述和次要内容。
- **三级文字**（`#8a8f98`）：柔和灰色，用于占位符、元数据和弱强调内容。
- **四级文字**（`#62666d`）：最柔和的文字——时间戳、禁用状态、微妙标签。

### 品牌与强调色
- **品牌靛蓝**（`#5e6ad2`）：主要品牌色——用于 CTA 按钮背景、品牌标识和关键交互表面。
- **强调紫**（`#7170ff`）：更亮的变体，用于交互元素——链接、激活状态、选中项。
- **强调悬停**（`#828fff`）：更亮、更饱和的变体，用于强调元素的悬停状态。
- **安全薰衣草**（`#7a7fad`）：柔和的靛蓝，专门用于安全相关的 UI 元素。

### 状态色
- **绿色**（`#27a644`）：主要成功/激活状态。用于"进行中"指示器。
- **祖母绿**（`#10b981`）：次要成功——胶囊徽章、完成状态。

### 边框与分隔线
- **主要边框**（`#23252a`）：实色深色边框，用于显著分隔。
- **次要边框**（`#34343a`）：稍浅的实色边框。
- **三级边框**（`#3e3e44`）：最浅的实色边框变体。
- **微妙边框**（`rgba(255,255,255,0.05)`）：超微妙半透明边框——默认选项。
- **标准边框**（`rgba(255,255,255,0.08)`）：标准半透明边框，用于卡片、输入框、代码块。
- **线条色调**（`#141516`）：几乎不可见的线条，用于最微妙的分隔。
- **三级线条**（`#18191a`）：稍明显的分隔线。

### 浅色模式中性色（用于浅色主题场景）
- **浅色背景**（`#f7f8f8`）：浅色模式下的页面背景。
- **浅色表面**（`#f3f4f5` / `#f5f6f7`）：微妙的表面色调。
- **浅色边框**（`#d0d6e0`）：浅色场景下的可见边框。
- **浅色边框备选**（`#e6e6e6`）：更浅的备选边框。
- **纯白**（`#ffffff`）：卡片表面、高亮。

### 遮罩
- **主要遮罩**（`rgba(0,0,0,0.85)`）：模态/对话框背景——极暗，用于聚焦隔离。

## 3. 排版规则

### 字体家族
- **主要字体**：`Inter Variable`，备选字体：`SF Pro Display, -apple-system, system-ui, Segoe UI, Roboto, Oxygen, Ubuntu, Cantarell, Open Sans, Helvetica Neue`
- **等宽字体**：`Berkeley Mono`，备选字体：`ui-monospace, SF Mono, Menlo`
- **OpenType 特性**：全局启用 `"cv01", "ss03"`——cv01 提供备选小写 'a'（单层），ss03 调整特定字母形态以获得更干净的几何外观。
